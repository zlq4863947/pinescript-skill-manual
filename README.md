## PineScript是什么

**PineScript** 是[TradingView](https://www.tradingview.com/) 设计的脚本开发语言。它允许用户创建自定义指标并在他们的服务器上解释执行。Pine 被设计为一种轻量级语言，专注于开发指标和策略。TradingView 的大部分内置指标已被放入 **PineScript** 中。
